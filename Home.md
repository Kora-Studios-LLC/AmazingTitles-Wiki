# AmazingTitles Home

Welcome to the official documentation home for AmazingTitles.

AmazingTitles is a Spigot and Paper plugin focused on animated text delivery across titles, subtitles, action bars, boss bars, smart notifications, interactive chat messages, extensions, and developer-facing API features.

## Why AmazingTitles

- built-in animation system for common title effects
- extension support for custom animation packs
- interactive message utilities for chat-based UX
- SmartBar notification support
- compatibility range from `1.8.8` to `1.21.11`
- public API for custom integrations and developer tools
