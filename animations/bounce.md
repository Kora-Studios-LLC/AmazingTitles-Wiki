# BOUNCE

`BOUNCE` is the built-in two-color bounce animation. It creates a mirrored moving highlight and then reverses it.

## Arguments

- `Color1(Hex/Legacy)`
- `Color2(Hex/Legacy)`

## Notes

- this is the default built-in bounce
- additional bounce-themed variations can be supplied by extensions

## Example

```text
/at sendAnimation all p:TITLE,d:4,fps:14 BOUNCE #FFAA00 #FFFFFF &6Season reset
```
