# FADE_IN_WRITER

`FADE_IN_WRITER` reveals text one character at a time while showing a writer symbol at the end.

## Arguments

- `Writer(Single Word)`

## Example

```text
/at sendAnimation all p:TITLE,d:4,fps:15 FADE_IN_WRITER | &bCreating world
```

## Tip

Use a short writer symbol such as `|`, `_`, or `.` for the cleanest result.
