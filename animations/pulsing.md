# PULSING

`PULSING` fades text between two colors over time.

## Arguments

- `Color1(Hex/Legacy)`
- `Color2(Hex/Legacy)`

## Notes

- works best on RGB-capable versions
- useful for premium title screens and status highlights

## Example

```text
/at sendAnimation all p:TITLE,d:4,fps:20 PULSING #00AAFF #FFFFFF &bLoading profile
```
