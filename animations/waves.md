# WAVES

`WAVES` animates a two-color sweep that moves across the text and back.

## Arguments

- `Color1(HEX/Legacy)`
- `Color2(HEX/Legacy)`

## Best use cases

- action bars
- loading states
- rotating announcements

## Example

```text
/at sendAnimation all p:ACTIONBAR,d:5,fps:12 WAVES #00AAFF #FFFFFF &bSynchronizing...
```
