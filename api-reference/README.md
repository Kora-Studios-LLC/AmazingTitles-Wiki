# API Reference

This section is the developer-facing reference area for AmazingTitles.

If you are integrating with the plugin from another module, extension, or custom feature, start here.

## Core concepts

- `AmazingExtension` for extension jars
- `AnimationBuilder` for custom animation registration
- `AmazingTitles` static API helpers for runtime actions
- `ComponentArguments` for delivery configuration

## Namespace

All current imports should use:

```text
org.korastudios.amazingtitles
```
