# Changelog

This page tracks notable documentation-level milestones for the current AmazingTitles codebase.

## 6.0-SNAPSHOT

Current working snapshot in this repository.

Highlights:

- namespace migrated to `org.korastudios.amazingtitles`
- support added for `1.8.8` through `R1_8_R3`
- `R1_21_R1` aligned for `1.21.11` Paper usage
- update checker moved from Spigot to GitHub Releases
- extension set expanded with new themed extension modules
- local wiki created and reorganized for GitBook usage

## Previous project lineage

AmazingTitles existed previously under the older namespace and documentation style. This repository now documents the current Kora Studios codebase direction.
