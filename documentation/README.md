## Documentation map

AmazingTitles documentation is split into a few practical layers:

- setup and compatibility
- commands and runtime usage
- built-in animation behavior
- extension loading and integration hooks
- public API reference for developers

That keeps the GitBook navigation clean while still matching the real plugin structure.
